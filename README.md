# Backend with Nest & Typescript
## Installation of the backend project

Execute the following command to execute backend project
## QUICK START BACKEND
cd .\backend\
npm install
npm run start:dev
## Installation

```bash
$ npm install
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
